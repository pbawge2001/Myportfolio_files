AUTOMATIC FLOOR CLEANER ROBOT USING - IOT

OBJECTIVE, The objective of this project is to design a Bluetooth controlled Robot for cleaning the floor by giving instructions from android Mobile. It is very useful for cleaning the floors.By using Bluetooth module, we can direct and turn the system as the user needs. It works great and controlled manually based on the user convenience.Definitely makes cleaning easier and merrier while enabling anyone to build something rather than buy.In modern days interior decorations are becoming an important role in our life.

Requirements:-
  • Arduino Uno.
  • ATMEGA328P Microcontroller.
  • HC-05 Bluetooth Model.
  • Channel Relay.
  • Arduino IDE

Advantages:-
  • Battery operated floor cleaning machine.
  • Convenient product.
  • Using bluetooth technology.
  • Controlling robot through android smart phone.
